## Engines & Platforms

### 1. VeriTech Verification Engine (CVK 1100)

The core verification system that powers all document authentication and validation services.

**Features:**
- 10-Level Verification Standard
- Document authenticity validation
- Chain of custody verification
- Legal admissibility assessment
- Court-ready evidence packaging

**Verification Levels:**
| Level | Description | Use Case |
|-------|-------------|----------|
| 1-3 | Basic verification | Initial document review |
| 4-6 | Standard verification | Case assessment |
| 7-9 | Advanced verification | Legal proceedings |
| 10 | Maximum verification | Court submission |

### 2. Investigation Engine

The intelligence gathering and analysis system for complex investigations.

**Capabilities:**
- Entity relationship mapping
- Financial trail analysis
- Corporate structure investigation
- Asset discovery
- Witness management
- Evidence chain documentation

**Investigation Protocols:**
- OSINT (Open Source Intelligence)
- Financial forensics
- Corporate registry analysis
- Property registry searches
- Court record analysis

### 3. Kavan AI Website

The client-facing platform for vulture fund victims in UK & Ireland.

**Services:**
- Document verification services
- Case assessment (€200/£170 Stage 1)
- Full legal pack preparation (€2,000/£1,700 Stage 2)
- Witness coordination
- Legal referral network

## Operational Philosophy

### Human-in-the-Loop

All Orb AI systems operate on a **mandatory human-in-the-loop principle**:
- Human beings lead all operations
- AI provides backup and works in partnership
- Critical decisions require human approval
- All outputs are human-reviewed before delivery

### Quality Standards

All outputs must meet the **VeriTech Standard**:
- Court-admissible evidence quality
- Forensic accountancy methodology for financial assessments
- Legal and regulatory compliance
- Professional documentation standards

## Getting Started

```bash
# Clone the repository
git clone https://github.com/darrylkavanagh-ux/Veritech.git

# Navigate to the project
cd Veritech

# Install dependencies
pnpm install

# Run verification engine
pnpm run verify

# Run investigation engine
pnpm run investigate
```
