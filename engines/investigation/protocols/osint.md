## 4. Documentation Requirements

### Source Documentation

Every piece of information must be documented with:
- Source URL or reference
- Date accessed
- Screenshot/PDF capture
- Verification status
- Confidence level

### Chain of Custody

- Who collected the information
- When it was collected
- How it was stored
- Who has accessed it

### Evidence Packaging

For court-ready evidence:
1. Create exhibit reference
2. Include source documentation
3. Provide verification certificate
4. Maintain chain of custody log
